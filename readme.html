    Installation

    Requred Elixir ~> 1.6 (Installing Elixir)

    Install dependencies 

    mix deps.get

    Open interactive shell 

    iex -S mix

    Run module/function 

    iex> PatternMatching.sum([1, 2, 3], 0)
6

    Tests 

    mix test
